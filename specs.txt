text exactly. 'Leg Press' and 'leg press' are treated as two different exercises by Google Sheets.

Essential Formulas
These six formulas do 90% of the work in the tracker. You don't need to understand how they work — just
copy them, then change the quoted exercise name and column references to match yours.
4.1  Pull the Last Logged Weight for Any Exercise
Place this on a Day Template tab in the 'Last Weight' column next to each exercise. It searches the Log
tab and returns the most recent weight entry for that exercise.
=IFERROR(
  INDEX(Log!E:E,
    MATCH(2, 1-(Log!D:D="Leg Press"), 0)
  ),
  "No data"
)
# Change 'Leg Press' to any exercise name
# Change Log!E:E to Log!G:G for Set 2 weight, Log!I:I for Set 3
■  Copy this formula down your entire exercise list, changing the name in quotes each time. Use Ctrl+H (Find & Replace) to batch-update exercise names.
4.2  Total Volume for a Session Today (Sets × Reps × Weight)
This multiplies weight × reps for every matching session row today and sums them. Place it on the
Dashboard.
=SUMPRODUCT(
  (Log!C:C="Legs") *
  (Log!A:A=TODAY()) *
  Log!E:E * Log!F:F
)
# Swap 'Legs' for any session name
# Swap TODAY() for a specific date: DATE(2025,5,1)
# Add more set columns to include Set 2 and 3 volume:
# + Log!G:G*Log!H:H + Log!I:I*Log!J:J
4.3  Count Training Sessions This Week

Your Personal Gym & Physio Tracker Guide  ·  Page 8
=COUNTIFS(
  Log!A:A, ">="&(TODAY()-WEEKDAY(TODAY(),2)+1),
  Log!A:A, "<="&(TODAY()-WEEKDAY(TODAY(),2)+7),
  Log!C:C, "<>Rest"
)
# WEEKDAY(TODAY(),2) returns 1=Mon through 7=Sun
# Subtracting it from TODAY() gives last Monday
# Adding 7 gives next Sunday — the full current week
4.4  Personal Best (Heaviest Weight Ever) for One Exercise
=MAXIFS(Log!E:E, Log!D:D, "Leg Press")
# Returns the highest value in column E (Set 1 Weight)
# where column D (Exercise) equals 'Leg Press'
# Repeat for every exercise on your Dashboard PR table
# Change Log!E:E to Log!G:G to check Set 2 weight PRs
4.5  Days Since Last Physio Session
Place this on your Dashboard. It shows how many days ago you last completed physio — helping you see
at a glance if you're overdue.
=IFERROR(
  TODAY() - MAX('Physio Log'!A:A),
  "No sessions logged"
)
# 'Physio Log'!A:A is the date column in your Physio Log tab
# MAX returns the most recent date
# TODAY() minus that = days elapsed
# Add conditional formatting: red if value > 2 (overdue)
4.6  Next Physio Due Date
=IFERROR(
  MAX('Physio Log'!A:A) + 2,
  "Log your first session"
)
# Takes the last physio date and adds 2 days
# Format this cell as a Date (Format > Number > Date)
# Pair with conditional formatting:
#   Green  if >= TODAY()  (on track)
#   Red    if <  TODAY()  (overdue)

8.2  Compliance Formula (Column H)
This formula automatically flags whether each session was completed on time (within 2 days of the
previous one).
# In cell H3 (start from H3, not H2 — needs a previous row to compare):
=IF(A3="", "",
  IF(A3-A2 <= 2,
     "On time",
     "Late (" & (A3-A2) & " days gap)"
  )
)
# Drag this formula down column H for all rows
# It compares each session date to the previous one
# Shows 'On time' if gap is 2 days or less
# Shows 'Late (4 days gap)' etc. if you missed sessions

Progressive Overload Tracking
Progressive overload is the most important training principle. Your tracker should make it obvious when to
add weight — no mental arithmetic needed.
The +2.5kg Next Target Column
# On each Day Template tab, column E next to 'Last Weight':
=IF(D2="—", "—", D2 + 2.5)
# D2 is your Last Weight formula cell
# This always shows the next weight to attempt
# For lighter exercises (curls, extensions), use + 1.25 instead
Rep-Based Progression for Bodyweight Exercises
# For Abs and Calf Raise — track reps, not weight
# Add a 'Last Reps' formula on the Day Template tab:
=IFERROR(
  INDEX(Log!F:F, MATCH(2, 1-(Log!D:D="Abs"), 0)),
  "—"
)
# Next target reps in the adjacent cell:
=IF(D2="—", "—", D2 + 2)
# When you hit the top of your rep range (e.g. 30 reps),
# add weight (weighted crunch/weighted calf raise)
# and reset the rep target to the bottom of the range
Deload Detection — vs Last Session Column
# On each Day Template tab, add a 'vs Last' column:
=IF(OR(D2="—",E2="—"), "—",
  IF(E2>D2, "+" & (E2-D2) & "kg",
  IF(E2<D2, "-" & (D2-E2) & "kg",
  "Same")))
# D2 = last session weight, E2 = session before that
# (requires formula 4.1 variations for previous sessions)
# Apply conditional formatting:
# Value starts with '+' → green fill (improved)
# Value = 'Same'        → yellow fill (plateaued)
# Value starts with '-' → red fill  (regressed)
# Two+ consecutive red cells = time to deload or review recovery

■■  If you see red on the same exercise for 3+ sessions in a row, check sleep, nutrition, and whether you need a deload week (reduce weight 40%, keep reps the same). This is normal every 6–8 weeks.

Tips & Best Practices
 Tip
 Detail
Log between sets, not after
the gym
Open Google Sheets on your phone after each exercise. Memory fades fast —
even 30 minutes later you'll mis-remember weights.
Keep units consistent
Choose kg or lbs on Day 1 and never switch. If you do switch, add a note in the
Notes column for that date so your charts don't show a false jump.
Lock your Day Template tabs
Right-click a Day Template tab → Protect sheet → Restrict editing. This stops
accidental formula deletion.
Use the Google Sheets mobile
app
Install the free app — your tracker syncs across all devices automatically. Pin it to
your home screen.
Back up monthly
File → Download → Microsoft Excel (.xlsx) once a month. Store in Google Drive
or your phone's camera roll.
Don't build everything on Day
1
Start with just the Log tab and Day Templates. Add the Dashboard and charts
after 2–3 weeks of data. They become meaningful once you have history.
Review weekly — Sunday 5
min rule
Every Sunday, scan your Log. Identify any exercises stalled for 2+ weeks and
plan a strategy: increase weight, increase reps, or swap the exercise.
Physio before or after training
Log physio as a separate entry in the Physio Log — don't mix it with your workout
Log entries. Keep the two trackers clean and separate.
Set a phone reminder for
physio
Set a recurring reminder every 2 days: 'Physio due — check tracker'. Pair it with a
habit anchor (e.g. always do it right after your morning coffee).
You now have a complete, personalised gym and physio tracker.
Start with Section 1 — create the file and the tabs. Then build the Log tab (Section 2)
and add the Physio Log (Section 8). Log one full training cycle before touching the
Dashboard or charts.
The data tells the story. Your job is to collect it consistently. Now go lift.
